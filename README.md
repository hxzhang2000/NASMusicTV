# NAS Music TV

一款开源的 Android TV 端 NAS 音乐播放器，让电视上的歌词显示不再是奢望。

## 功能特性

### 音乐播放
- **NAS 音乐播放**：连接 Jellyfin / Navidrome 后端，播放 NAS 中的音乐
- **网络音乐搜索**：通过 Meting-API 在线搜索网易云音乐，独立于 NAS 后端
- **网络歌曲播放**：302 重定向解析真实 mp3 URL，播放链接实时解析不缓存
- **播放模式**：顺序播放、单曲循环、列表循环、随机播放
- **播放队列管理**：添加、移除、清空、移动排序
- **播放队列持久化**：重启应用自动恢复上次播放队列
- **均衡器**：预设方案 + 频段增益调节

### 网络音乐
- **在线搜索**：支持中文关键词搜索，多端点自动 fallback（Mikus / Redcha / Qijieya）
- **网络歌词**：自动获取在线歌词，失败回退到酷狗/网易云
- **网络封面**：Coil 自动跟随 302 重定向显示封面
- **歌曲收藏**：收藏网络歌曲，下次可直接播放（LRU 上限 500 条）
- **端点配置**：设置页可选择预设端点或自定义 Meting-API 端点

### 歌词系统
- **逐行滚动歌词**：适配电视大屏
- **逐字高亮**：支持卡拉 OK 逐字变色效果，50ms 高频刷新平滑过渡
- **歌词匹配**：自动从网络匹配歌词并本地缓存
- **多来源获取**：MP3 内嵌、本地缓存、本地文件、网络匹配，支持来源切换
- **高亮模式记忆**：逐字/逐行选择跨页面切换保留

### 曲库浏览
- **搜索功能**：拼音首字母匹配（"zjl" 搜索 "周杰伦"）+ 子串匹配
- **艺术家 / 专辑 / 歌曲**：网格卡片浏览，点击进入详情页
- **流派 / 年代浏览**：按流派和出版年份筛选
- **收藏与最近播放**：支持收藏、播放次数统计
- **播放列表**：创建、删除、播放、管理歌曲

### 封面与显示
- **专辑封面**：支持内嵌元数据提取和在线获取
- **艺术家封面**：自动获取艺术家图片
- **多封面轮播**：歌曲/专辑/艺术家多张封面每 10 秒轮播，单张时静态显示，加载失败自动 fallback
- **网络歌词联动封面**：NAS 歌曲切换到"在线歌词"时同时获取并加入网络封面轮播
- **歌词来源标签**：显示当前歌词来源（内嵌/本地/网络/在线），点击切换

### TV 体验
- **遥控器优化**：完整的 D-Pad 导航和焦点系统
- **HDMI-CEC**：支持电视遥控器媒体键控制
- **前台通知**：媒体播放通知栏，支持 play/pause/next/previous
- **多架构支持**：ARM64 / ARMv7 / x86_64

## 技术栈

- Kotlin + Jetpack Compose for TV
- Media3 / ExoPlayer 音频引擎
- OkHttp 网络层
- Coil 图片加载
- TinyPinyin 拼音转换（兼容 API 22+）

## 界面预览

- **播放页**：左侧专辑封面 + 右侧歌词，进度条 + 控制按钮，网络歌曲显示 NET 标识
- **曲库页**：专辑 / 艺术家 / 歌曲 / 流派 / 年代 / 收藏 / 最近播放 / 网络
- **网络音乐页**：在线搜索歌曲，收藏列表展示，收藏/队列操作
- **详情页**：专辑详情、艺术家详情（歌曲列表 + 播放全部）
- **队列页**：播放队列管理（删除、移动、清空）
- **设置页**：主题、动画、播放模式、缓存管理、均衡器、关于、网络音乐端点配置（左侧导航栏支持遥控器滚动）
- **连接页**：Jellyfin / Navidrome 服务器配置

## 构建与运行

```bash
# 设置 JAVA_HOME
export JAVA_HOME="C:\Program Files\Android\Android Studio\jbr"

# Debug APK
./gradlew assembleDebug

# Release APK
./gradlew assembleRelease

# 运行测试
./gradlew test

# 推送到电视（将 192.168.0.116 替换为你的电视 IP）
adb -s 192.168.0.116:5555 install -r app/build/outputs/apk/debug/app-debug.apk
```

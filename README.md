# NAS Music TV

一款开源的 Android TV 端 NAS 音乐播放器，让电视上的歌词显示不再是奢望。

## 功能特性

### 音乐播放
- **NAS 音乐播放**：连接 Jellyfin / Navidrome 后端，播放 NAS 中的音乐
- **播放模式**：顺序播放、单曲循环、列表循环、随机播放
- **播放队列管理**：添加、移除、清空、移动排序
- **均衡器**：预设方案 + 频段增益调节

### 歌词系统
- **逐行滚动歌词**：适配电视大屏
- **逐字高亮**：支持卡拉 OK 逐字变色效果
- **歌词匹配**：自动从网络匹配歌词并本地缓存
- **多来源获取**：MP3 内嵌、本地缓存、本地文件、网络匹配，支持来源切换

### 曲库浏览
- **搜索功能**：拼音首字母匹配（"zjl" 搜索 "周杰伦"）+ 子串匹配
- **艺术家 / 专辑 / 歌曲**：网格卡片浏览，点击进入详情页
- **流派 / 年代浏览**：按流派和出版年份筛选
- **收藏与最近播放**：支持收藏、播放次数统计
- **播放列表**：创建、删除、播放、管理歌曲

### 封面与显示
- **专辑封面**：支持内嵌元数据提取和在线获取
- **艺术家封面**：自动获取艺术家图片
- **歌词来源标签**：显示当前歌词来源（内嵌/本地/网络），点击切换

### TV 体验
- **遥控器优化**：完整的 D-Pad 导航和焦点系统
- **HDMI-CEC**：支持电视遥控器媒体键控制
- **前台通知**：媒体播放通知栏，支持 play/pause/next/previous
- **多架构支持**：ARM64 / ARMv7 / x86_64

## 技术栈

- Kotlin + Jetpack Compose for TV
- Media3 / ExoPlayer 音频引擎
- OkHttp 网络层
- Coil 图片加载

## 界面预览

- **播放页**：左侧专辑封面 + 右侧歌词，进度条 + 控制按钮
- **曲库页**：专辑 / 艺术家 / 歌曲 / 流派 / 年代 / 收藏 / 最近播放
- **详情页**：专辑详情、艺术家详情（歌曲列表 + 播放全部）
- **队列页**：播放队列管理（删除、移动、清空）
- **设置页**：主题、动画、播放模式、缓存管理、均衡器、关于
- **连接页**：Jellyfin / Navidrome 服务器配置

## 构建与运行

```bash
# 设置 JAVA_HOME
export JAVA_HOME="C:\Program Files\Android\Android Studio\jbr"

# Debug APK
./gradlew assembleDebug

# Release APK
./gradlew assembleRelease

# 运行测试
./gradlew test
```
